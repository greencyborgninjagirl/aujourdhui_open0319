## 二、内测链接生成方式

### 方式 A：局域网内测（推荐，手机与电脑同一 WiFi）

1. **获取电脑局域网 IP**
   - macOS：系统设置 → 网络 → Wi-Fi → 详情 → IP 地址（如 `192.168.1.100`）
   - 或终端执行：`ifconfig | grep "inet " | grep -v 127.0.0.1`

2. **配置 API 地址**
   - 复制 `aujourdhui-frontend/.env.test.example` 为 `aujourdhui-frontend/.env.development.local`
   - 将 `192.168.1.100` 改为你的电脑 IP
   - （构建内测包则复制为 `.env.test`，执行 `npm run build:h5 -- --mode test`）

3. **启动后端（局域网可访问）**
   ```bash
   chmod +x run-api-lan.command
   ./run-api-lan.command
   ```

4. **启动前端（开发模式，局域网可访问）**
   ```bash
   cd aujourdhui-frontend
   npm run dev:h5
   ```
   - 终端会显示类似：`Local: http://localhost:8080/` 和 `Network: http://192.168.x.x:8080/`
   - **内测链接**：用手机浏览器打开 `http://你的电脑IP:8080`（如 `http://192.168.1.100:8080`）

5. **或使用构建产物**
   ```bash
   cd aujourdhui-frontend
   npm run build:h5 -- --mode test
   npx serve -s dist/build/h5 -l 8080
   ```
   - 手机访问 `http://你的电脑IP:8080`

### 方式 B：公网内测（部署到云服务器）

- 将前端构建产物部署到 Vercel / 腾讯云静态托管 / 阿里云 OSS 等
- 将 API 部署到云服务器（如 Railway、腾讯云、阿里云等）
- 在 `.env.test` 中设置 `VITE_API_BASE=https://你的API域名`
- 构建：`npm run build:h5 -- --mode test`
- 分享前端部署后的 URL 给测试者

## 三、中国大陆访问注意事项

- **牌面图片**：来自 `upload.wikimedia.org`，部分地区可能加载较慢或需代理
- **名画链接**：部分指向海外博物馆（artic.edu、metmuseum.org 等），链接可访问，图片加载取决于网络
- 若牌面图无法加载，可考虑后续将图片镜像到国内 CDN
