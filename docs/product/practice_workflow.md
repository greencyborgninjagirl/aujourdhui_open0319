# aujourd'hui 交互与抽牌规则

## 1. 抽牌仪式感
1. **静心**：引导用户深呼吸，集中精神在自己身上，尽量在安静环境下准备开始抽牌。
2. **提问**：采用开放式问题（如：“我今天需要知道什么？”），拒绝封闭式“是/否”提问。
3. **洗牌**：随机混合 78 张牌（含正逆位）。

## 2. 抽牌模式
- **单牌日运**：抽取一张，代表当日整体能量。

## 3. 解读原则
- **正逆位辨析**：正位代表外在显现，逆位代表能量受阻或内在心理。
- **直觉优先**：鼓励用户先看图，再看文字解读。
- **量子视角**：解读仅为当下时间线上的一种可能性，运势掌握在用户手中。
